Projects

Prescription Online

An app to help keep records of all prescriptions. Uses core vision to read words from the picture taken, which is then converted into words and placed on the correct category which is then placed into a core data database.

The purpose of this app is not only to make the lives of people easier by having to worry less about losing prescriptions or being overwhelmed by having too many prescription papers, but also to limit the amount of trees being cut down which will be better for the environment.

iExpress

A machine learning progrem dedicated to detect facial expressions. Uses Computer Vision and machine learning technology such keras and CNN to evaluate pixels from the picture which are placed in a matrix to determine what facial expression the person on the picture has. The results are then put onto a confusion matrix to evalute which expressions were detected correctly by the model and which ones the model had trouble predicting.

See Live Source Code

Pan-O-Rama

A progrem dedicated to stitch together different picture in order to create a panorama. Uses computer vision technology such as laplacian blending and perspective wraping to find pixels that are the same between the two picture to help stitch the images together. After fidning similar pixels, uses RANSAC to transform and map the pixels so that the pictures overlap and creates a panorama.
